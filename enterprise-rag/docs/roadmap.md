# 演进路线图

## MVP

- 单租户或弱租户隔离基础版
- 文件上传、异步解析、切片、Embedding、混合检索、问答
- 基础控制台、最小审计、最小观测

## 一期

- 完整多租户、RBAC、知识库级和文档级权限
- 私有化部署方案定型，支持内网交付
- Provider 可插拔，兼容本地模型和外部兼容接口
- 引用溯源、反馈闭环、离线评测、索引重建
- 运维能力补齐：限流、告警、死信重试、配额管理

## 二期

- 增加更多数据连接器，如 SharePoint、Confluence、数据库、企业 IM
- 引入更强的治理能力，如审批流、内容质检、成本分析
- 支持多模态文档、图谱增强检索、跨知识库联合问答
- 支持更细粒度的 Prompt/策略实验平台

## 长期方向

- 从知识库问答扩展到企业知识中台
- 引入工作流编排、Agent 协同和工具调用，但仍以可审计为前提
- 构建统一评测平台和租户级优化建议系统
